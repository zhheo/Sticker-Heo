## 快速上手

### Twikoo

Twikoo可以在json文件中添加：[查看文件](/twikoo.json)

### Valine

[查看文件](/valine.json)

### Waline
在emoji选项中添加配置项：
```js
 emoji: [
    'https://github-do.panbaidu.cn//https://raw.githubusercontent.com/zhheo/Sticker-Heo@latest/Sticker-100/',
]
```
